# 两数相加


给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

    输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
    输出：7 -> 0 -> 8
    原因：342 + 465 = 807


题解：

    这题思路很简单就是遍历两个链表，把值相加

    然后把进位的值保存，每次相加都加上

    如果一个链表遍历完了，就算 0

    注意pre的值如果链表遍历完了还大于0 就要加到结果上面，循环条件加个pre > 0 就ok
