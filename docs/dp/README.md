# 动态规划实战


## 动态规划解题的思路

1. 递归+记忆化 = 递推 (可以先尝试递归，再推成dp)
2. 状态的定义
3. 状态转移方程
4. 最优子结构 (前面 n-1 个状态有一个自由子结构可以推导出 n状态)


## 模板

```
 function run() {


     let dp = [] // 定义状态

     dp[0] = 1 // 初始状态 可能给 -1状态

     for ... // 递推 0 - n - 1 可能出现 -1 - n 

     return dp[n] // 返回最优值

 }

```
